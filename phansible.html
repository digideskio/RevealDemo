¿Qué es Phansible?

Configurador web para crear entornos virtuales de desarrollo

Creado para entornos de desarrollo PHP


Objetivo

Yo programo con WAMP

... y yo uso MAMP

... pues yo tengo una máquina virtual

... hasta el infinito y más allá con mi VIM en producción

¿Problema?

Entornos de desarrollo totalmente diferentes

Muy difícil de mantener

Sistema excesivamente tradicional


¿Qué podemos hacer?

Simple... vamos a trabajar todos igual

Requisitos

Los requisitos para que funcione Phansible son:

Virtualbox
Ansible
Vagrant

:: Mac OS X y GNU/Linux ::

:: Windows ::

Currently Ansible can be run from any machine with Python 2.6 or 2.7 installed (Windows isn’t supported for the control machine) *Workaround*

Aplicación para la gestión de máquinas virtuales

En nuestro caso será el "provider" por defecto

Vagrant

Software para la creación y configuración de entornos virtuales de desarrollo

Puede comunicarse con varios "providers" y diferentes CM (Configuration Management)

Configuration Management para automatizar la configuración de máquinas virtuales

Para que funcione en Windows hay que tirar de cygwin
