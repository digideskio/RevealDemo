¿Qué es Log.io?

Logs en tiempo real en tu propio navegador

Sin tener que acceder a los servidores

Varios tipos de logs: Apache, Aplicación, Sistema...

De repente un error salvaje apareció

Componentes

Powered by node.js y socket.io

Web Server - Servidor web donde vemos los logs

Log.io Server - Servidor que recolecta los logs para presentarlos en el web server

Harvester - Agente que se comunica con el Log.io Server para proporcionarle los logs

El rol de Harvester puede estar en local o en servidor remotos así que podemos ver logs de otros servers
